Provide reasoning for the extracted data from a previous step based on the provided `AI Instructions`, `Reference Documents`, and `Results from Previous Extraction`. Your output MUST be a JSON ARRAY of objects, where each object corresponds to a result in the `Results from Previous Extraction`, maintaining the SAME ORDER. *   **extractedValue**: A detailed explanation of the extraction logic used to obtain the `Results from Previous Extraction`, referencing specific sections or keywords from the `Reference Documents` where applicable. *   **confidenceScore**: An integer between 0 and 100 representing the AI's confidence level in the extraction. This score should reflect the clarity of the relevant information in the `Reference Documents` and the certainty of the extraction logic.
*   **documentSource**: A string referencing the specific source document and page number (if applicable) from the `Reference Documents` from which the `extractedValue` was derived. Be as specific as possible (e.g., "DocumentA, page 3", "Section 2.1 of DocumentB").

Return multiple items as an array, maintaining the SAME ORDER as the input `Results from Previous Extraction` to ensure proper record linkage.
